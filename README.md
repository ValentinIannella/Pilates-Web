# Pilates Web - Valsy Studio

Web application for managing and booking pilates class appointments.

# # 🚀 Description
This project allows users to select a date and time to book a pilates class.

The system validates availability and prevents double bookings.

# # 🛠️ Technologies
- HTML
- CSS
- JavaScript

# # ⚙️ Features
- Date selection
- Time slot selection
- Availability validation
- Appointment booking

# # ▶️ How to use
1. Open the `index.html` file
2. Select a date and time
3. Confirm the booking
